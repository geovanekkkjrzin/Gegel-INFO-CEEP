Glossário de Geovane

Este Glossario Tem Como Função Instruir Você, Estudante de informatica

Servidor WEB
Um servidor web é um conjuntos de computadores que forma "ele", ou seja o servidor WEB, ele armazena, processa e entrega arquivos de sites. Varios tipos de servidores web fornecem conteudo estatico e dinâmico ao navegador.
Servidor DNS
O Servidor DNS convertes os nomes dos sites para endereço ip, Assim sendo póssivel ser lida na Linguagem dos computadores
Dominio
Dominio é um nome utilizado para localizar conjuntos de computadores na internet
Hospedagem
Hospedagem é um lugar onde podemos colocar os sites, como se fosse um aluguel, pagamos para deixar os sites onlines, sem risco de queda
